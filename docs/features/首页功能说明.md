# 知秋工作平台 - 首页功能说明

## 📖 概述

全新设计的首页采用 Fluent Design 风格，提供直观、美观且高效的用户体验。首页集成了欢迎横幅、数据统计、快速访问、最近使用和使用提示等多个功能模块。

---

## ✨ 主要特性

### 1. 🎨 智能欢迎横幅
- **动态问候语**：根据当前时间自动显示不同的问候语
  - 早上好（5:00-12:00）☀️
  - 中午好（12:00-14:00）🌤️
  - 下午好（14:00-18:00）⛅
  - 晚上好（18:00-22:00）🌙
  - 夜深了（22:00-5:00）✨

- **实时时钟**：显示完整的日期和时间信息
- **渐变背景**：根据主题自动切换渐变色彩
  - 浅色模式：天蓝色 → 粉色渐变
  - 深色模式：钢青色 → 紫色渐变
- **淡入动画**：页面加载时平滑的淡入效果

### 2. 📊 数据概览统计
展示平台使用情况的关键数据：
- **总功能数**：显示平台提供的功能总数
- **使用次数**：统计工具使用频率
- **处理文件**：记录已处理的文件数量
- **节省时间**：计算为用户节省的工作时间

每个统计卡片包含：
- 图标标识
- 数据标题
- 核心数值
- 增长趋势（本周/本月）

### 3. 🚀 快速访问区域
提供常用功能的快捷入口，包括：
- 数据叠加：计算矢量套合占比
- 字段分离：按字段分离要素
- DXF转换：提取DXF图层
- SHP转KMZ：转换为奥维格式
- 影像拼接：多影像拼接处理
- 获取中心点：获取要素中心坐标

特点：
- 卡片式设计，一目了然
- 悬停效果增强交互体验
- 流式布局，自适应窗口大小

### 4. 📝 最近使用记录
显示用户最近操作的功能列表，包括：
- 功能图标
- 功能名称
- 使用时间（相对时间显示）

方便快速再次访问常用功能。

### 5. 💡 使用提示
提供实用的使用技巧和更新信息：
- **快捷搜索**：介绍搜索功能的使用方法
- **批量处理**：说明批量处理能力
- **最新更新**：展示版本更新内容

## 🎯 设计理念

### 视觉设计
- **Fluent Design**：采用微软 Fluent Design 设计语言
- **渐变效果**：使用柔和的渐变色彩提升视觉吸引力
- **卡片布局**：信息模块化，层次分明
- **响应式**：自适应不同窗口尺寸

### 交互设计
- **动画效果**：淡入动画提升加载体验
- **悬停反馈**：鼠标悬停时提供视觉反馈
- **可点击元素**：快速访问和最近使用卡片支持点击交互
- **实时更新**：时钟每秒更新，保持信息新鲜

### 功能设计
- **一站式入口**：首页汇集所有关键信息
- **数据驱动**：通过数据展示使用情况
- **快速导航**：提供多种方式快速访问功能
- **智能提示**：主动提供使用建议

### 自定义绘制
使用 QPainter 实现渐变背景：
```python
def paintEvent(self, e):
    painter = QPainter(self)
    gradient = QLinearGradient(...)
    # 根据主题设置不同渐变色
    gradient.setColorAt(0, color1)
    gradient.setColorAt(1, color2)
    painter.fillPath(path, gradient)
```

## 🎨 主题适配

首页完全支持浅色/深色主题切换：

### 浅色模式
- 横幅渐变：天蓝色(#87CEEB) → 粉色(#FFB6C1)
- 卡片背景：白色系
- 文字颜色：深色系

### 深色模式
- 横幅渐变：钢青色(#4682B4) → 紫色(#8A2BE2)
- 卡片背景：深色系
- 文字颜色：浅色系

主题切换实时响应，无需重启应用。
