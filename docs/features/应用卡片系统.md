# 应用卡片系统使用说明

## 📋 概述

应用卡片系统将数据处理功能整合成小卡片形式展示，支持搜索和弹窗操作。

## 🎯 功能特点

1. **卡片展示**：所有功能以卡片形式在应用界面展示
2. **搜索功能**：通过标题栏搜索框快速查找功能
3. **弹窗操作**：点击卡片后以独立窗口展示具体功能
4. **分类管理**：功能按类型分组展示

## 📁 文件结构

```
知秋工作平台/
├── app_card_interface.py    # 应用卡片展示界面
├── app_functions.py          # 应用功能管理器
├── demo.py                   # 主程序（已集成）
└── 数据处理.py              # 原数据处理功能
```

## 🚀 使用方法

### 1. 查看应用

启动程序后，点击左侧导航栏的"应用"，即可看到所有功能卡片。

### 2. 搜索应用

在标题栏搜索框输入关键字（如"裁剪"、"转换"等），卡片会自动过滤显示。

### 3. 打开功能

点击任意卡片，会弹出独立窗口展示该功能的完整界面。

## ⚙️ 技术实现

### AppCard 类
```python
- 功能：单个应用卡片组件
- 属性：app_id, icon, title, description
- 信号：clicked(str) - 发送应用ID
```

### AppCardInterface 类
```python
- 功能：应用卡片展示界面
- 方法：
  - initCards() - 初始化卡片
  - filterCards(keyword) - 搜索过滤
  - onCardClicked(app_id) - 处理点击
```

### AppFunctionManager 类
```python
- 功能：应用功能管理器
- 方法：
  - openApp(app_id, parent) - 打开应用弹窗
```

## 🔧 扩展开发

### 添加新功能

1. 在 `app_card_interface.py` 的 `initCards()` 方法中添加卡片定义：
```python
('new_func_id', FIF.ICON, '功能名称', '功能描述'),
```

2. 在 `app_functions.py` 的 `openApp()` 方法中添加处理逻辑：
```python
elif app_id == 'new_func_id':
    title = "功能名称"
    widget = YourFunctionWidget()
```

### 自定义卡片样式

修改 `AppCard` 类的样式参数：
- `setFixedSize()` - 调整卡片大小
- `setContentsMargins()` - 调整内边距
- `setSpacing()` - 调整间距

## 📝 注意事项

1. **导入问题**：确保所有功能模块正确导入
2. **弹窗大小**：默认1200x800，可根据功能调整
3. **搜索关键字**：支持中文标题和描述搜索
4. **性能优化**：卡片采用延迟加载，避免启动卡顿
