# ⚡ 快速参考手册

## 🚀 启动应用

```bash
# 方式1: 双击运行
启动应用.bat

# 方式2: 命令行
python demo.py
```

## 🎨 主题设置

| 操作 | 路径 | 生效 |
|------|------|------|
| 切换浅色 | 设置 → 个性化 → 应用主题 → 浅色 | ⚡ 立即 |
| 切换深色 | 设置 → 个性化 → 应用主题 → 深色 | ⚡ 立即 |
| 自动主题 | 设置 → 个性化 → 应用主题 → 跟随系统 | ⚡ 立即 |
| 自定义色 | 设置 → 个性化 → 主题色 → 选择颜色 | ⚡ 立即 |

## 🔧 界面调整

| 设置项 | 可选值 | 生效方式 |
|--------|--------|----------|
| 界面缩放 | 100%, 125%, 150%, 175%, 200%, 自动 | 🔄 重启 |
| 语言 | 简体中文, English, 自动 | 🔄 重启 |
| Mica效果 | 开/关 (仅Win11) | ⚡ 立即 |
| 模糊半径 | 0-40 | ⚡ 立即 |

## 📝 快捷操作

### 恢复默认设置
```bash
1. 关闭应用
2. 删除 config.json
3. 重新启动
```

### 手动编辑配置
```bash
1. 关闭应用
2. 编辑 config.json
3. 重新启动
```

### 高分屏设置
```
设置 → 个性化 → 界面缩放 → 150% → 重启
```

## 🎯 常用配置组合

### 护眼模式
```json
{
    "ThemeMode": "Dark",
    "ThemeColor": "#ff4a9d00",
    "AcrylicBlurRadius": 20
}
```

### 高性能模式
```json
{
    "MicaEnabled": false,
    "AcrylicBlurRadius": 0
}
```

### 专业模式
```json
{
    "ThemeMode": "Light",
    "ThemeColor": "#ff0078d4",
    "DpiScale": 1.25
}
```

## 🐛 问题速查

| 问题 | 解决方案 |
|------|----------|
| Mica 是灰色 | 需要 Win11 系统 |
| 缩放不生效 | 需要重启应用 |
| 语言不变化 | 需要重启应用 |
| 配置丢失 | 删除 config.json 重启 |
| 亚克力警告 | `pip install PyQt6-Fluent-Widgets[full]` |
